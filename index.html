MyMisu OS

a bare-metal x86 operating system, running in your browser
Click "Start" to boot the OS

How to use

  1. Click Start MyMisu OS and wait ~10 seconds
  2. Click inside the black screen to capture your keyboard
  3. Login with the credentials below
  4. Type help to see every command

Press ESC outside the screen to release keyboard control.

What you're seeing

Real bare-metal OS running in JavaScript via v86. No Linux underneath. The kernel was written from scratch in C and x86 assembly. Real preemptive scheduler, real syscalls via int 0x80, real filesystem.
